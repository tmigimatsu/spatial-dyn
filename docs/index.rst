spatial-dyn
===========

.. _spatial-dyn: https://github.com/tmigimatsu/spatial-dyn

`spatial-dyn`_ is a C++/Python library for simulating basic articulated bodies
using Featherstone's efficient dynamics algorithms using spatial vectors.
